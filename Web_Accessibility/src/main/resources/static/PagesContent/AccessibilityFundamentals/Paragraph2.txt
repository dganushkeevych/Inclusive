Інклюзивний дизайн - дизайн основних продуктів та / або послуг, які доступні та використовуються якомога більшою кількістю людей без потреби в спеціальній адаптації або спеціалізованому дизайні.
Інклюзивний дизайн не передбачає, що завжди можна (або доречно) розробити один продукт для задоволення потреб усього населення. Натомість забезпечує створення дизайну шляхом:
- розробки сімейства продуктів та похідних для забезпечення якнайкращого охоплення - використання продукту великою кількістю різноманітного населення;
- забезпечення того, щоб кожен окремий товар не мав чітких та цільових користувачів;
- зниження рівня можливостей, необхідних для використання кожного продукту, з метою покращення взаємодії з користувачем для широкого кола споживачів у різних ситуаціях.
Інклюзивний дизайн - це процес. Доступність - це результат.
