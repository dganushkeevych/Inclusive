Accessible Rich Internet Applications (ARIA) — це простий набір атрибутів HTML, які визначають способи зробити веб-контент і веб-програми (особливо ті, що розроблені за допомогою Ajax, JavaScript і новіших веб-технологій, як-от Bootstrap), більш доступними для людей з обмеженими можливостями. Наприклад, ARIA забезпечує доступні навігаційні орієнтири, віджети JavaScript, підказки у формі та повідомлення про помилки, живе оновлення вмісту тощо.

ARIA має набір спеціальних атрибутів доступності HTML, які можна додати до будь-якої розмітки, але особливо підходить для HTML. Атрибут role визначає конкретну роль для типу об’єкта (наприклад, стаття, сповіщення, повзунок або кнопка). Додаткові атрибути ARIA надають інші корисні властивості, такі як опис форми або поточна ширина індикатора виконання. Атрибути ARIA також можна використовувати для визначення активного чи вимкненого стану об’єктів (особливо кнопок).
