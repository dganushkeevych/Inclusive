Тег alt або альтернативний (alt) атрибут html в основному використовується для опису зображення. У HTML це виглядає так:

<img src =”image-location” alt=”тут іде опис вашого зображення”>
Навіть якщо ви використовуєте візуальний інструмент, який приховує ваш HTML-код для створення вебсайту, ви завжди матимете можливість ввести опис зображення.

Звичайно alt може бути порожнім, але правильно надавати кожному елементу корисний опис для програм зчитування з екрана.
Якщо б ви були сліпими, то alt=«Жінка» не є дуже зрозумілим, а «Жінка малює блок-схему дизайну, включаючи доступність, зручність використання, брендинг і дизайн» є набагато кращим описом.
