Будь ласка, для веб-доступності не возьтеся з тегами html за умовчанням. Використовуйте кнопки для кнопок, а не прив’язки для кнопок. Щоб прив’язки працювали так само, як кнопки, вам потрібно додати додаткову подію javascript, що є дуже дурним підходом, коли ви можете легко використовувати звичайну голу кнопку. Використовуйте кнопки для кнопок, прив’язки для посилань, «table, tbody, td, th» для таблиць, «h1, h2, h3, …» для заголовків тощо.

Порушення з ними повністю заплутає програми зчитування з екрана, оскільки всі програми зчитування з екрана використовують певний спосіб читання елементів на веб-сторінці.
